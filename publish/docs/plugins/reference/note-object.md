---
title: "Note Object"
date: 2023-05-23
---
**NOTE**: this object is subject to change 

The note object is a JSON object that contains 3 parameters: `title`, `content`, `source`. It might look like:
```
{
	"title": "title of the note",
	"content": "content within the note",
	"source": "a source",
}
```
